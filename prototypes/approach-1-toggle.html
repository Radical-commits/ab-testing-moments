Approach 1: A/B Toggle — Minimal Disruption

A simple toggle inside the Message Composition step enables A/B testing. When turned on, the editor splits into two variant tabs and a config panel appears. The rest of the broadcast flow stays unchanged. A/B testing feels like an optional add-on to the existing message step.

+ Easy to build + Minimal learning curve + No extra steps in non-test campaigns - A/B testing feels like an afterthought - Config buried inside message step
